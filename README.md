# electionPKU-浏览器插件

*请不要用本退课机刷课，否则会受到严厉处分（退光你的课）*

***本项目作为浏览器扩展工作**，自动识别并填充验证码，代码摘自[PKUElectiveHelper](https://github.com/goudanZ1/PKUElectiveHelper)。*

## 使用方法：

1. **导入浏览器扩展**

   - 克隆仓库 / 从Releases下载&解压ZIP 

   - 在edge/chrome的“加载已解压的扩展程序”中选择此目录，添加扩展

2. **注册识图账户**

   - 本项目使用第三方工具识别验证码，在[TT识图](http://www.ttshitu.com/)注册并登录后充值识图账号1元即可，识图为 0.002 元/次

   - 使用文本编辑器查看目录下的inject.js，按照代码中的要求与格式填入你的TT识图账号（重启扩展以应用修改）

3. **打开 PKU 选课网的“补退选”页面，就能看到~~脚本已经生效啦~~你的课程被退光啦~~~**
